# 🎁 The Birthday Gauntlet

A beautifully interactive, gamified trivia app built with Python and Streamlit. This app serves as a custom birthday puzzle, requiring the player to successfully answer five unique memory-testing questions based on personal relationship milestones to unlock their real-world birthday gift.

## 🌟 Features
* **Five Custom Levels:** Covers relationship milestones from the first date to future travel goals.
* **Smart UI Components:** Integrates a seamless calendar date-picker for precise chronological questions.
* **Over-the-Top Celebrations:** Features heavy, chained native animations (`st.balloons()` and `st.snow()`) that trigger multiple cascading celebration waves upon entering correct answers.
* **Persistent Session States:** Keeps track of player progression and dynamic hints without losing data on page refreshes.

🚀 Local Quickstart
If you want to run this application locally inside an active Anaconda or standard Python virtual environment, follow these quick terminal commands:

Navigate to the directory containing the file:

Bash
cd Desktop
Launch the application through Python's module router:

Bash
python -m streamlit run birthday_game.py
